# Hand Tracking Cursor

Control your system cursor using hand tracking via the webcam.

## Requirements

- Python 3.8+
- Webcam

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate   # Windows
# source .venv/bin/activate  # macOS/Linux
pip install -r requirements.txt
```

## Usage

```bash
python -m hand_tracking_cursor
```
