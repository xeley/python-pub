# Hand Landmarker model

Place the MediaPipe Hand Landmarker model file here so the app can find it.

**Option A – run the download script (recommended):**

```powershell
python scripts\download_hand_landmarker.py
```

**Option B – manual download:**

1. Download: https://storage.googleapis.com/mediapipe-models/hand_landmarker/hand_landmarker/float16/1/hand_landmarker.task  
2. Save it in this folder as: `hand_landmarker.task`

Final path: `models/hand_landmarker.task`
